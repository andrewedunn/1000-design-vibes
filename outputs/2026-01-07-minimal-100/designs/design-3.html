What is Modern Lux?

Modern Lux is a sophisticated knowledge base platform built for developers who value both aesthetics and functionality. We combine Scandinavian design principles with modern development practices to create documentation that's a pleasure to read and easy to navigate.

Design Philosophy

Our approach centers on three core principles that guide every decision:

  • Clarity First ‚Äî Information architecture designed for immediate comprehension and effortless navigation
  • Luxurious Simplicity ‚Äî Generous whitespace and refined typography create a premium reading experience
  • Developer-Centric ‚Äî Code examples, API references, and technical details presented with precision and care
üí° Why Choose Modern Lux?

Traditional documentation often sacrifices aesthetics for utility, or vice versa. Modern Lux proves you don't have to compromise. Our Scandinavian-inspired design creates a calm, focused environment that enhances comprehension while maintaining the technical depth developers need.

Getting Started

Installation

Begin your journey with Modern Lux using your preferred package manager. The installation process is straightforward and takes less than a minute:

npm install modern-lux --save

# or using yarn
yarn add modern-lux

# or using pnpm
pnpm add modern-lux

Basic Configuration

Configure Modern Lux by creating a lux.config.js file in your project root. The configuration system uses sensible defaults while remaining fully customizable:

export default {
  theme: 'scandinavian',
  colorMode: 'light',
  typography: {
    heading: 'Playfair Display',
    body: 'Inter'
  },
  navigation: {
    position: 'sidebar',
    searchEnabled: true
  },
  content: {
    maxWidth: '900px',
    codeHighlighting: true
  }
}

Creating Your First Page

Modern Lux uses markdown with extended syntax for rich documentation. Create a new file in your docs/ directory:

---
title: "Your First Page"
description: "A warm introduction"
category: "Getting Started"
---

# Welcome to Your Documentation

Start writing beautiful docs immediately.
No complex setup required.
