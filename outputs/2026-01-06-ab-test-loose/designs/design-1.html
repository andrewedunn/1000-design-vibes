Global Shift

Neo brutalist-inspired design for children

Design System Foundation

Typography

Headings: IBM Plex Mono (monospace)

Body: Roboto Slab (slab serif)

Scale: Minor third (1.111)

Style: Uppercase headings, tight letter-spacing

Color Theory

Primary: Monochromatic grays

Accents: Warm & cool tones

Saturation: Desaturated palette

Mood: Primary with subtlety

Spatial Design

Base Unit: 12px spacing

Density: Ultra-airy (2-3x spacing)

Radius: 24px organic corners

Shadows: Hard (4px offset, 0 blur)

Component System

Floating Containers

Hard shadows create depth without glass blur effects. Each floating element has a 4px offset shadow, organic corner radius (24px), and responds to interaction with subtle translation. The glassmorphic paradigm merges with brutalist honesty—raw geometry softened for play.

Grid Hierarchy

Three-column justified layout creates visual rhythm. Typography-driven hierarchy uses size and weight rather than color. Monospace headings in uppercase anchor content blocks. Whitespace becomes the primary design tool—generous margins let elements breathe.

Interactive States

All buttons and cards have focus indicators with 3px outline (4.5:1 contrast). Hover states translate elements -2px, amplify shadows. Active states compress. Touch targets exceed 44px minimum. Keyboard navigation fully supported with visible focus throughout.

Expanded Component Library

Monospace Headers

IBM Plex Mono creates technical, raw aesthetic. Uppercase treatment reinforces graphic weight. Tight letter-spacing (-0.02em) creates cohesion. Perfect for a children's design that doesn't condescend—honest, structured, playful.

Accent Colors

Warm accent (#d4a574) and cool accent (#5a8a99) provide visual interest within desaturated palette. Used as duotone gradients on featured cards. Low overall contrast maintains calm, child-friendly environment while guiding attention.

Organic Geometry

24px border-radius softens brutalist edges. Not fully rounded (pill-shaped), but organic enough to feel inviting. Contrast with 2px borders that reinforce structure. This balance defines the aesthetic: honest structure with gentle character.

Density Approach

Ultra-airy spacing (3-4x base unit between major sections) creates breathing room. Perfect for children—less cognitive load, clearer focus areas. Whitespace isn't wasted; it's intentional hierarchy. Each element floats independently.

Hard Shadows

4px offset, 0 blur radius creates sharp definition. No atmospheric perspective—elements feel present and tactile. Responds to interaction: hover amplifies (6px offset), active compresses (2px offset). Shadow as interactive feedback.

Accessibility

4.5:1 minimum contrast ratio throughout. 44px minimum touch targets on all buttons. Visible focus indicators (3px outline, 2px offset). Supports prefers-reduced-motion. Keyboard navigation fully functional. WCAG AA compliant.

Design Principles

Honest Structure

Neo-brutalism reveals the structure. Borders visible, spacing logical, hierarchy clear. No visual tricks or fake materials. Children appreciate honesty—this design respects their intelligence without pretension.

Gentle Playfulness

Floating animations, organic corners, and duotone gradients soften the brutalist edge. Movement is subtle (4s float cycle), not jarring. Colors desaturated but present. The design feels approachable to children, not intimidating.

Whitespace as Tool

Ultra-airy density means every element matters. No clutter, no distraction. Whitespace creates focus and reduces cognitive load. Perfect for healthcare context—clarity and calm matter. Children thrive in uncluttered environments.

Design System

Global Shift demonstrates neo-brutalist principles adapted for children's healthcare applications. Raw structural honesty meets gentle playfulness.
