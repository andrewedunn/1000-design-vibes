Component Library

A thoughtfully crafted collection of components inspired by the Arts & Crafts movement. Each element is designed with honest construction, natural materials, and functional beauty in mind.

Button Interactive

Buttons trigger actions and communicate available interactions. They feature visible borders and tactile hover states that mimic the feeling of pressing physical buttons.

Live Preview
HTML 
<button class="btn btn-primary">Primary Action</button>
<button class="btn btn-secondary">Secondary</button>
<button class="btn btn-success">Success</button>
<button class="btn" disabled>Disabled</button>
CSS Variables 
--color-primary: #8B6F47;
--color-surface: #FFFFFF;
--border-width: 2px;
--radius-default: 2px;
--space-sm: 8px;
--space-lg: 24px;
Prop Type Default Description
variant string default Button style: primary, secondary, success, warning, error
disabled boolean false Disables button interaction and applies disabled styling
size string medium Size variant: small, medium, large
onClick function - Click event handler function

Input Form

Text inputs allow users to enter and edit text. Strong borders and clear focus states provide confidence in form interaction.

Live Preview
HTML 
<input
  type="text"
  class="input"
  placeholder="Enter your name"
  aria-label="Name input"
/>
Prop Type Default Description
type string text Input type: text, email, password, number, etc.
placeholder string - Placeholder text displayed when input is empty
disabled boolean false Disables input interaction

Card Container

Cards group related content and actions. Honest borders and subtle shadows create natural depth without excessive decoration.

Crafted with Care

Every component in this library reflects the Arts & Crafts philosophy of honest construction and functional beauty. No element exists without purpose.

  <h3 class="card-title">Crafted with Care</h3>
  <p class="card-content">
    Every component reflects honest construction...
  </p>
</div>
Prop Type Default Description
elevation string low Shadow depth: low, medium, high
padding string medium Internal spacing: small, medium, large
