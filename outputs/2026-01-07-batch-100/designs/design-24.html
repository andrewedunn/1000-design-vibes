Design Philosophy

Block Hub combines brutalist directness with Swiss International precision, creating a calm yet powerful design system for enterprise real estate. The tetradic color palette—warm muted reds, greens, golds, and slates—provides extreme contrast while maintaining visual harmony. Organic corners soften the brutalist edges, creating an approachable aesthetic.

Core Attributes: Brutalist UI + Swiss Grid | Tetradic Warm Muted Colors | Humanist Sans Headings + Slab Serif Body | Balanced Density | Organic Corners | Layered Shadows

Typography System

Major third scale (1:1.25) creates clear visual hierarchy. Uppercase headings with humanist sans-serif convey authority. Slab serif body text provides warmth and readability with tight line-height and loose letter-spacing.

Spacing & Patterns

All spacing based on 10px unit with major third scale multipliers (1, 1.4, 2, 2.8, 4, 5.6). Organic 16px border radius on all containers, 8px on components, 24px on large surfaces.

Accessibility & Standards

This design system meets WCAG AA standards with minimum 4.5:1 contrast ratios across all text. All interactive elements feature visible focus indicators, and touch targets meet or exceed 44px minimum size. Supports prefers-reduced-motion user preference.
