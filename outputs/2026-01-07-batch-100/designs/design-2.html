Design System Overview

Aesthetic

Claymorphic UI with warm, desaturated pastels. Organic border radius and dashed accent lines create a psychedelic developer-friendly aesthetic.

Principles

Monospace typography ensures code readability. Medium shadows with warm tints add depth. Dense asymmetric layout balances visual hierarchy through position and color.

Grid

Asymmetric columns with mixed alignment. Flexible spacing base (10px) adapts to diverse layouts.

Color Palette

Warm, desaturated pastels with monochrome neutrals. Accent hierarchy through subtle saturation shifts.
