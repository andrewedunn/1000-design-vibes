Modern Tide

An editorial bauhaus design system with cyber sensibility and calm aesthetic. Structured, information-dense layouts with monochromatic high-contrast palette and fabric texture detail.

color system

Monochromatic palette with balanced saturation. High contrast between text and surfaces ensures readability. Neutral temperature creates objective, trustworthy presence.

typography

Headings: Inter (Humanist Sans)
editorial power

Inter provides clean, geometric forms. Small-caps treatment emphasizes structure and creates visual rhythm through text.

Body: Crimson Text (Transitional Serif)

Crimson Text brings warmth and readability to long-form content. Very loose line height creates breathing room in dense layouts, improving scannability.

grid & spacing

Base unit: 4px. Dense, modular grid adapts to content. Justified alignment emphasizes orderly editorial flow.

design features

  • Border Style: 2px medium borders with consistent weight
  • Radius: 12px pill-shaped interactive elements

design principles

Bauhaus Structure: Form follows function with geometric precision. Editorial Authority: Typography-driven hierarchy emphasizes content. Cyber Objectivity: Monochromatic neutrality suggests impartiality. Calm Sophistication: Generous spacing and measured pacing create serene experience. Swiss Grid: Objective, structured layouts with mathematical rigor. Fabric Texture: Subtle grain suggests craftsmanship. Accessibility First: High contrast and large type support seniors audience.
