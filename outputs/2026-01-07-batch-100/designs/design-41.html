Typography System

Indie Flower for Headings

Handwritten serif provides warmth and personality. Letter spacing sits at 2px for dramatic impact.

Quicksand body text maintains readability while supporting the organic aesthetic.

Component System

Organic Shapes

Cards feature rounded corners and organic radius values (24px) that soften the interface while maintaining structure.

Dramatic Shadows

Deep shadows with extended offsets create floating elevation. Shadows use dark colors for maximum impact.

Balanced Density

12px spacing base provides breathing room between elements while maintaining visual cohesion across sections.

Accent Lines

Thin gradient lines at card tops reference craft traditions and guide visual attention naturally.

Design Principles

Handcrafted Warmth

Every element celebrates the human hand. Handwritten fonts, warm metallic colors, and organic borders create an approachable premium aesthetic.

Extreme Contrast

High contrast between backgrounds and metallic accents ensures visual hierarchy and accessibility while maintaining visual drama.

Energetic Movement

Dramatic shadows and hover states encourage interaction. The design feels alive, perfect for fitness and wellness applications.
