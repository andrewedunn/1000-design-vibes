Body Text

This is standard body text using the Bebas Neue display typeface. The loose line height and letter spacing create an airy, readable experience that honors the art nouveau aesthetic while maintaining modern legibility.

This is small text for captions and secondary information. It maintains the same elegant character while being appropriately scaled for supporting content.

Caption text is used for metadata and timestamps.

Text can contain inline links that use dashed underlines as a nod to the border style. Links transition smoothly to solid on hover.

Muted text is used for less prominent information that shouldn't compete with primary content.
